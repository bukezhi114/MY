# 快捷键管理器

一个纯前端的快捷键管理工具，用于记录和快速查询各软件的快捷键。

## 功能

- **预设快捷键库**：内置 311 条常用软件快捷键（VS Code、IntelliJ IDEA、Word、Excel、Photoshop、Blender、OBS 等）
- **快速搜索**：按应用名、快捷键组合、功能描述实时搜索
- **分类筛选**：按功能类别过滤
- **增删改查**：支持手动添加/编辑/删除快捷键
- **导入导出**：JSON 格式备份和恢复

## 使用方式

直接在浏览器中打开 `index.html` 即可使用，无需安装任何依赖。

## 数据存储

数据保存在浏览器 localStorage 中，可通过导出功能备份为 JSON 文件。
